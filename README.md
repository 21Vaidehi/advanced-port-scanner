# Multithreaded Port Scanner Inspired by Nmap

A Python-based multithreaded TCP port scanner developed for learning cybersecurity, networking, and socket programming concepts.

## Features

- Detects open TCP ports
- Supports custom port ranges
- Fast multithreaded scanning
- Saves results to file
- Colored terminal output
- Service identification

## Technologies Used

- Python
- Socket Programming
- ThreadPoolExecutor
- Colorama

## How to Run

```bash
pip install colorama
python scanner.py
```

## Example Output

[OPEN] Port 80 | HTTP
[OPEN] Port 443 | HTTPS
